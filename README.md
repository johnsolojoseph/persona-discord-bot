# Discord Persona Bot

## Command List

1. p!
    registers user into database
2. p!help
    send help text to channel
3. p!encounter
    generates a random encounter for the whole server
4. p!info {name}
    show info card on persona
5. p!negotiate
    talk to persona to join your party
6. p!reset
    reset user progress
